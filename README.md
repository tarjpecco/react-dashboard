# flexcomply

## Pre Requirement

- Node: > 8.x
- Npm: > 6.x
- Yarn: > 1.9.x

## Install Guide

- Nodemon global install
`npm install -g nodemon`

## Run Guide

- Local Environment
`yarn start:local`
- Dev API Environment
`yarn start:api`
- Master Environment
`yarn build`

## ESlint
- `yarn eslint`

## Configuration
- Frontend port: 3000
#### Backend URL
- Local Environment
`localhost:5000`
-  Dev API Environment
`https://google.com`
-  Master Environment
`https://google.com`
